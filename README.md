# Installation
```
git clone https://github.com/edfloreshz/tasks.git
cd tasks
sudo just install
```

# Build
```
git clone https://github.com/edfloreshz/tasks.git
cd tasks
cargo build
```

## Dependencies
- [libcosmic](https://github.com/pop-os/libcosmic?tab=readme-ov-file#building)
